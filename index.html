Formula Reference
Demand Forecast
= 0.7 × current sales + 0.3 × previous sales
If no previous data: forecast = current sales
Daily Demand
= forecast / salesWindowDays
Cycle Demand = dailyDemand × scheduleDays
Shipment [Push]
= max(0, forecast × 1.2 − current WH inventory)
If region action = STOP → shipment = 0
Shipment [Pull]
= max(0, reorderPoint − current WH inventory)
Reorder point = expected demand per cycle
Procurement
= max(0, totalForecast × 1.3 − availableStock)
availableStock = hub + all regional inventory
Sell-Through
= sales / inventory  ·  target 0.65–0.85
